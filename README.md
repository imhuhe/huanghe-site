# huanghe-site

黄河 / HUANGHE — personal brand site.

Vite + React + TypeScript + Tailwind CSS + Framer Motion.

## Develop

```bash
npm install
npm run dev
```

Dev server runs at http://localhost:5180.

## Build

```bash
npm run build
```

Static output goes to `dist/`. Deployed to Netlify (config in `netlify.toml`).
